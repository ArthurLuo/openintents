The OpenIntents ConvertCSV is an extension for OI shopping list
that lets you import and export the lists to CSV files.

To obtain the current release, visit
  http://www.openintents.org

---------------------------------------------------------
release: 1.0.0
date: 2009-?

- First public release on Android SDK 1.1.

Features: 
- Delete all notes from OI Notepad.
- Delete all lists from OI Shopping list.
